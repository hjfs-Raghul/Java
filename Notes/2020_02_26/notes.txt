Servlet & JSP

	servlet creation (Servlet Class)
		
		doPost
			method = "post" -> calls doPost method inside the servlet class mentioned in the action attribute
		doGet
			method = "get"/by default -> calls doPost method inside the servlet class mentioned in the action attribute
		service
			by default , service method will be called first , irrespective of the method attribute,
			inside the servlet class mentioned in the action attribute
